# rsk-stats

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload 
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

```

## Icons


- Install vsvg : 
```
npm install vsvg -g
```

- Copy svgs on : static/svg/icons
- Compile svgs
 ```
  npm run svg
 
 ```
